# 1) Bending stress calculation of simply supported I-beam
# Problem Statement
Simply supported beam, L = 2.0 m, P = 10 kN at center. I-section: h=100 mm, b=50 mm, t_web=t_flange=5 mm.
(i)   Draw Free body Diagram (Freikörperdiagramm)
(ii)  Find reaction forces (Reaktionen)
(iii) Plot shearforce (Querkraft) and Bendingmoment (Biegemoment) Diagrams
(iv)  Calculate Moment of Inertia I about neutral axis
(v)   Find Maximum Bending Stress (Biegespannung) σ_max
(vi)  Verify σ_max with Roarks
